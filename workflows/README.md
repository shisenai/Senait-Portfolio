# Workflow & Pipeline Examples

## 1. Hydrologic Data QA/QC Workflow

```text
Raw Monitoring Data
        ↓
Data Validation
        ↓
Reference Station Comparison
        ↓
Quality Flagging
        ↓
Reporting
        ↓
Modeling & Analysis
```

**Tools:** GVA, DBHYDRO, SQL, Excel
