# Environmental Monitoring Dashboard

Power BI dashboard developed for water resources and environmental monitoring.

## Dashboard Preview

![Environmental Monitoring Dashboard](../Images/environmental%20dashboard.PNG)

## Key Features

- Average Streamflow KPI
- Average Water Level KPI
- Average Water Quality KPI
- Monthly Streamflow Trend Analysis
- Water Quality vs Water Level Relationship
- Interactive Date Filtering

## Tools Used

- Power BI
- Excel
- Environmental Analytics
- Data Visualization
